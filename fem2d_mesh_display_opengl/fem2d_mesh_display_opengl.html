FEM2D_MESH_DISPLAY_OPENGL
Display a Polygonal Mesh Using OpenGL

FEM2D_MESH_DISPLAY_OPENGL is a C++ program which reads the name of two data files defining a polygonal mesh, reads the data, and displays the mesh using OPENGL.

The files defining the mesh consist of one file containing the coordinates of the points, and a second file that lists the indices of the nodes making up each element. All the elements in the mesh must have the same order, that is, the number of vertices.

Usage:

fem2d_mesh_display_opengl prefix
where prefix is the common filename prefix:
  • prefix_nodes.txt, the node coordinates;
  • prefix_elements.txt, the element definitions.
reads the node data from nodes.txt, the element data from elements.txt and displays a plot of the mesh.

Examples and Tests:

ELL is an L-shaped region covered by triangles.

  • ell_nodes.txt, the node coordinates.
  • ell_elements.txt, the nodes that form each element.
  • ell.png, a PNG image of the mesh.

RECT is a rectangular region covered by quadrilaterals.

  • rect_nodes.txt, the node coordinates.
  • rect_elements.txt, the nodes that form each element.
  • rect.png, a PNG image of the mesh.

Q4 is a rectangular region covered by quadrilaterals, with a hole.

  • q4_nodes.txt, the node coordinates.
  • q4_elements.txt, the nodes that form each element.
  • q4.png, a PNG image of the mesh.

HEX is a mesh of hexagons.

  • hex_nodes.txt, the node coordinates.
  • hex_elements.txt, the nodes that form each element.
  • hex.png, a PNG mage of the mesh.
