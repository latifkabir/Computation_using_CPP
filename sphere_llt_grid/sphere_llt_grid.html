SPHERE_LLT_GRID
Latitude/Longitude Triangle Grids on a Sphere

SPHERE_LLT_GRID is a C++ library which constructs a grids of triangles using latitude and longitude lines over the surface of a sphere in 3D.

Languages:

SPHERE_LLT_GRID is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

  • SPHERE_LLT_GRID_LINES: latitude/longitude triangle grid lines.

  • SPHERE_LLT_GRID_POINT_COUNT counts points for a latitude/longitude grid.
